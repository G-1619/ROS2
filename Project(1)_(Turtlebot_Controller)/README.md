# ROS2 Project(1) "Turtlebot Controller"
PyQt를 이용해 만든 UI로 Gazebo 상의 Turtlebot을 제어한다.


## Requirements
* ROS2 Humble
* PySide6
* TurtleBot3 Packages
* Python (3.10.12)

## Usage
'''bash
git clone https://github.com/G-1619/ROS2/tree/c38f157ed8a8863b24e26774af2eeb4e7d4ae171/Project(1)_(Turtlebot_Controller)
colcon build
'''
